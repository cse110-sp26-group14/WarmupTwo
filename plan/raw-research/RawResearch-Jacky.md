## **Domain Research:**

### **Common Features**

Core Mechanics:

- Reels & Symbols  
- Paylines  
- Randomization  
- Spin Button  
- Bet System

Feedback Systems:

- Visual  
  - Animated Spinning Reels  
  - Highlighted Winning Paylines  
  - Flashing or Glowing Symbols

- Audio  
  - Reel spinning sounds  
  - win/loss sound effects

- Text/UI  
  - Display of credits, winnings, and bet size

## **User Research**

### **Persona 1: Casual Player**

Name: Timmy  
Age: 20  
Background: Freshly graduated college student that is employed  
Goals:

- Have a fun night at the casino  
- Quick and fun gameplay  
- Easy to understand rules 

Frustrations:

- Complicated interfaces  
- Too many features at once

Behavior:

- Plays in short sessions  
- Prefers simple controls and fast feedback

### **Persona 2: Competitive Player**

Name: Chad  
Age: 30  
Background: Unemployed, full-time gambler, has a lifetime positive winrates  
Goals:

- Maximize winnings  
- Understand game mechanics deeply

Frustrations:

- Lack of transparency in game logic  
- Repetitive Gameplay

Behavior:

- Plays longer sessions  
- Interested in advanced features like multipliers or bonuses

## **User Stories**

1. As a casual player, I want simple controls so that I can start playing immediately without confusion.  
2. As a casual player, I want clear visual feedback so that I can easily understand if I won or lost.  
3. As a player, I want to see my current balance and bet amount so that I can track my progress.  
4. As a competitive player, I want more advanced features (like multipliers or bonus rounds) so that the game feels engaging over time and easier to maximize winnings  
5. As a player, I want consistent rules so that I can learn and predict outcomes more effectively.
